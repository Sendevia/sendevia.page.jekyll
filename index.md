# 皮肤介绍

## 这是什么

这是一个 osu! 皮肤，它以游戏补丁的方式实现了对 osu! 全部界面元素的覆盖。  
它的设计风格扩展了《刀剑神域》第 1、2 季中操作界面的设计，结合更现代的设计风格，再将两者和谐相融。  

## 有什么优点

扩展了刀剑神域系列番剧中 UI 的设计  
拥有良好的打图手感  
简洁而美观的视觉体验  
通过修改 osu! 的资源 dll，实现了覆盖 osu! 的几乎所有界面元素  
定制化选项较为丰富
