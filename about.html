关于主题

简单介绍一下这个项目，这是一个 Material 3 (You) 风格的 Jekyll 主题，但我没有使用任何 UI 框架，例如mui。

这个主题原生实现了部分 Material 的组件样式，包括但不限于按钮、复选框、内容卡片，具体的使用范例在这里。

做这个主题的灵感来源主要是 material.io 官网，我去除了大部分但保留了一点，这样你才知道在用的是一款 Material 3 风格的主题。

主要功能

  1. 支持从头图中取色并应用颜色主题。这可以是动态取色吧
  2. 支持多种 Material 3 组件。但在 markdown 中用原生 html 真的好怪啊
  3. 响应式布局。是个现代网站都有吧
  4. 在移动端有接近原生应用的使用体验。虽然不是PWA
