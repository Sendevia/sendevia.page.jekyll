Buttons

概述 详细描述 Element Class 正常 正常，有图标 失效 失效，有图标
整个网站应使用统一的按钮样式，以保证视觉的统一性 Elevated buttons are essentially filled buttons with a lighter background color and a shadow. To prevent shadow creep, only use them when absolutely necessary, such as when the button requires visual separation from a patterned background. button elevated
The filled button’s contrasting surface color makes it the most prominent button after the FAB. It’s used for final or unblocking actions in a flow. filled
Filled tonal buttons have a lighter background color and darker label color, making them less visually prominent than a regular, filled button. They’re still used for final or unblocking actions in a flow, but do so with less emphasis. filled-tonal
Use an outlined button for actions that need attention but aren’t the primary action, such as See all or Add to cart. This is also the button to use for giving someone the opportunity to change their mind or escape a flow. outlined
Text buttons have less visual prominence, so should be used for low emphasis actions, such as an alternative option. text
